# 项目文档索引

本目录仅保留当前有效、可维护的工程文档。

## 文档清单

1. `产品需求说明书 (PRD).md`：产品功能与非功能需求、验收标准
2. `系统总体架构设计.md`：系统架构、当前闭环状态与下一步规划
3. `通信协议接口控制文档 (ICD).md`：协议字段与帧定义（权威来源）
4. `数据字典.md`：字段语义与取值范围
5. `FPGA 模块详细设计.md`：RTL 侧模块设计说明
6. `Python 模块详细设计.md`：上位机模块设计说明
7. `MA703FA_FPGAEncyclopedia.md`：项目快速总览（新人 30 分钟上手）
8. `MA703FA_APP_1G_UDP_PCS_Ethernet_SFP1_百科全书.md`：参考工程内部结构与使用说明
9. `protocol_contract_v1.json`：协议契约快照

## 维护规则

- 接口字段变更时，必须同步更新 ICD、数据字典和相关测试。
- 设计文档与当前源码不一致时，以当前源码和测试行为为准，并尽快回写文档。
- 临时过程文档不进入本目录长期维护范围。
