# 数据字典（小白版）

文档编号：FPGA-QT-01-003  
版本：V1.1  
日期：2026-05-30

## 1. 先看这三点

1. 本项目统一使用大端。
2. 价格类字段默认 float32。
3. 新增字段必须先入字典再写代码。

## 2. 基础行情字段

| 字段名 | 类型 | 说明 | 示例 |
|---|---|---|---|
| stock_code | char[8] | 股票代码+交易所 | 000858SZ |
| timestamp | uint32 | 秒级时间戳 | 1717036800 |
| open | float32 | 开盘价 | 123.45 |
| high | float32 | 最高价 | 124.10 |
| low | float32 | 最低价 | 122.90 |
| close | float32 | 收盘价 | 123.80 |
| volume | uint32 | 成交量（股） | 1500000 |

## 3. 指标字段（下行）

| 字段名 | 类型 | 说明 |
|---|---|---|
| MA5 | float32 | 5 周期均线 |
| MA10 | float32 | 10 周期均线 |
| RSI6 | float32 | 6 周期 RSI |
| RSI14 | float32 | 14 周期 RSI |

说明：M1.1 阶段允许这些字段先为占位值。

## 4. 决策字段

| 字段名 | 类型 | 取值 | 说明 |
|---|---|---|---|
| trade_signal | uint8 | 0/1/2 | 无/买/卖 |
| signal_strength | uint8 | 0-100 | 信号强度 |

## 5. 建议取值范围

| 字段 | 推荐范围 |
|---|---|
| open/high/low/close | 0.01 - 100000 |
| volume | 0 - 4294967295 |
| RSI6/RSI14 | 0 - 100 |
| signal_strength | 0 - 100 |

## 6. 与代码对应关系

- Python 协议定义：host_side/app/fpga_protocol.py
- FPGA 协议核心：fpga_side/rtl/src/m1_protocol_core.v
- 协议测试：host_side/tests/test_protocol.py

## 7. 新手自检清单

1. 字段类型是否匹配
2. 偏移是否匹配 ICD
3. 字节序是否大端
4. 测试是否通过
