# 产品需求说明书 \(PRD\)

**文档编号**：FPGA\-QT\-01\-001**版本**：V1\.1**日期**：2026\-05\-31**审批人**：刘浩

### 产品概述

本产品是一套基于 FPGA 硬件加速的股票量化交易原型系统，旨在利用 FPGA 的并行计算能力，实现技术指标的纳秒级实时计算，为高频量化交易提供硬件加速解决方案。系统通过 Python 上位机获取实时股票数据，经 UDP 协议发送至 FPGA 开发板进行高速计算，最终将计算结果回传至上位机生成买卖信号。

### 功能需求

#### 2\.1 数据获取功能

- 支持通过 AKShare 库获取 A 股股票实时 K 线数据

- 支持获取单只股票的 1 分钟、5 分钟、15 分钟 K 线数据

- 支持数据预处理，包括缺失值填充、异常值过滤

- 支持将 K 线数据按照协议格式封装成 UDP 数据包

#### 2\.2 硬件加速计算功能

- 支持 UDP 协议解析（物理层、IP 层、UDP 层）

- 支持 MA5、MA10 移动平均线计算

- 支持 RSI6、RSI14 相对强弱指数计算

- 支持计算结果的 UDP 协议封装与回传

- 支持数据校验与错误数据包丢弃

#### 2\.3 结果展示与决策功能

- 支持接收 FPGA 回传的计算结果

- 支持实时展示股票 K 线图与技术指标曲线

- 支持基于预设阈值生成买卖信号

- 支持信号强度显示（0\-100 分）

- 支持历史交易记录保存与查询

#### 2\.4 系统管理功能

- 支持 IP 地址与端口号配置

- 支持 FPGA 比特流文件烧录

- 支持系统运行状态监控

- 支持日志记录与错误排查

### 非功能需求

#### 3\.1 性能指标

- 单条 K 线数据处理延迟：≤1μs（从 FPGA 收到数据包到发出结果包）

- 系统吞吐量：≥1000 条 / 秒

- 指标计算准确率：≥99\.99%（与 Python 计算结果对比）

- 丢包率：≤0\.01%（在 1Gbps 网络环境下）

#### 3\.2 可靠性指标

- 连续运行时间：≥72 小时无故障

- 错误数据包处理率：100%（所有错误数据包均被正确丢弃）

- 数据一致性：FPGA 计算结果与 Python 计算结果完全一致

#### 3\.3 可维护性指标

- 代码注释覆盖率：≥30%

- 模块接口标准化：100%

- 文档完整性：100%

### 验收标准

| 编号 | 验收项 | 验收条件 | 验收方法 |
|---|---|---|---|
| AC-01 | 数据获取 | AKShare 成功拉取 A 股 1min/5min/15min K 线 | 运行 fetch_history.py，检查输出数据字段完整 |
| AC-02 | 数据预处理 | 缺失值填充、异常值过滤正确 | 运行 test_validator.py，注入异常数据验证 |
| AC-03 | 协议编解码 | 48B 上行/44B 下行帧，Big-Endian，CRC32 一致 | 运行 test_protocol.py + test_contract_snapshot.py |
| AC-04 | UDP 传输 | 收发正常，超时重试 ≤3 次 | 运行 test_udp_transport.py + test_mock_fpga_behavior.py |
| AC-05 | 协议核 | header/length/crc 校验，错误帧丢弃 | Vivado 仿真 tb_m1_protocol_core + tb_system_mixed |
| AC-06 | 指标计算 | MA/RSI/MACD/量比 指标输出 | Vivado 仿真 tb_indicator_top + tb_top |
| AC-07 | 评分决策 | 基于指标输出 0-100 评分 + 0/1/2 决策 | Vivado 仿真 tb_score_calc |
| AC-08 | 结果回传 | 结果帧字节流通过 udp_result_tx 打包 | Vivado 仿真 tb_udp_result_tx |
| AC-09 | 实时展示 | K 线图 + 技术指标曲线显示 | 上位机 GUI 手动验收 |
| AC-10 | 买卖信号 | 基于阈值生成交易信号 + 信号强度 | e2e_runner.py 端到端验证 |
| AC-11 | 系统管理 | IP/端口配置、日志记录、状态监控 | 配置文件检查 + 日志文件检查 |
| AC-12 | 稳定性 | 连续运行 ≥72h 无故障 | 长时压力测试（待执行） |
