基于 SFP 光口 UDP FPGA 股票高速交易项目8 人完整版详细工作任务

---
组员 1：上位机股票行情采集与数据预处理
主要工作任务
1. 基于现有股票 Python 代码，完善 FastAPI 后端服务、接口鉴权 Token 验证功能。
2. 对接 akshare 数据源，实现 A 股、港股、美股、ETF、LOF 多品种股票数据自动获取。
3. 完成股票代码合法性校验、时间区间参数处理、自动补全起止日期。
4. 对原始行情数据进行清洗：去重、删除空值、列名统一、数据类型转换、时间格式标准化。
5. 提取开盘、最高、最低、收盘、成交量原始 K 线数据，不做任何技术指标运算，只做规整封装。
6. 输出标准格式行情数据集，为后续 SFP 光口 UDP 封包提供规范数据源。
7. 配合组员 2 统一数据格式，保证上位机发包格式与 FPGA 解析格式完全对齐。
8. 【新增】 支持参数可配置下发（指标周期、评分阈值），通过 UDP 传给 FPGA 实时生效。
9. 【新增】 使用 Git 完成代码版本管理，提交记录可追溯。
交付成果
FastAPI 完整服务程序、多市场股票数据采集代码、数据清洗预处理模块、标准 K 线数据结构文档、Git 版本库、可配置参数接口。

组员 2：上位机 SFP 光口 UDP 通信协议与封包解包开发
主要工作任务
1. 牵头制定上下位机 SFP 光口 UDP 自定义通信协议：帧头、数据长度、股票数据域、校验和、帧尾格式定义。
2. 配置电脑 SFP 光口网卡 IP、子网掩码，与 FPGA 光口设置同一网段，保证二层链路互通。
3. 基于 Python 开发 SFP 光口 UDP 客户端程序，实现网络绑定、Socket 配置。
4. 将组员 1 整理好的股票原始 K 线数据，按照约定协议封装成二进制 UDP 数据包。
5. 实现通过 SFP 光口向 FPGA 发送 UDP 行情数据包。
6. 监听 SFP 光口 UDP 端口，接收 FPGA 回传的评分、交易信号、指标结果。
7. 对回传二进制数据进行解包、解析、还原成可读数值，供后续展示与比对。
8. 编写通信协议说明文档，统一格式标准供 FPGA 端开发参考。
9. 【新增】 增加丢包统计、CRC 错误统计、链路状态监控，日志输出便于调试。
10. 【新增】 支持下发配置帧（修改 MA 周期、RSI 周期、决策阈值）。
交付成果
上位机 SFP-UDP 收发程序、自定义通信协议文档、封包解包源码、网络配置说明、错误统计日志模块、配置帧协议扩展。

组员 3：米联客 MA FPGA 硬件工程搭建与 SFP 光口底层驱动

1. 新建米联客 MA FPGA 完整工程，完成工程目录规划、器件选型、全局时钟配置。

3. 配置 FPGA 千兆 / 万兆光口 PHY、MAC IP 核，完成底层以太网硬件链路初始化。
4. 配置 FPGA 端静态 IP、MAC 地址，完成与上位机 SFP 光口网络组网连通。
5. 搭建 FPGA 顶层框架，完成整体模块划分、端口定义、预留各子模块接口。
6. 调试光模块在位检测、光链路 Link 状态、高速串行数据收发正常。
7. 提供稳定可用的 FPGA 基础工程模板、约束文件、IP 核配置工程，供其他组员二次开发。
8. 【新增】 增加软复位模块、时钟锁定检测、GTP 状态监测，提升系统稳定性。
9. 【新增】 工程支持 Git 管理，目录规范化。
交付成果
FPGA 基础工程模板、SFP 光口底层驱动、管脚与时序约束文件、光口链路调通工程、硬件配置手册、复位与状态监测模块、规范化工程目录。

组员 4：FPGA SFP 光口以太网、UDP/IP 协议解析模块开发
主要工作任务
1. 基于组员 3 的 SFP 底层工程，开发 FPGA 以太网帧接收逻辑。
2. 实现 MAC 层帧解析、IP 头部解析、UDP 头部解析，剥离协议包头，提取纯股票业务数据。
3. 处理跨时钟域信号同步、字节对齐、位宽转换、数据流缓冲 FIFO 设计。
4. 加入帧头帧尾检测、CRC 校验、长度校验，过滤错误帧、干扰帧、丢包处理。
5. 把解析干净的原始股票 K 线数据，稳定传给后端指标运算模块。
6. 配合组员 8 编写模块 Testbench，参与 Modelsim 功能仿真，验证协议解析逻辑正确性。
7. 上板联调时排查光口数据错位、粘包、错包问题。
8. 【新增】 增加配置帧解析，支持上位机下发参数到指标 / 决策模块。
9. 【新增】 输出错误计数（错帧、丢包、CRC 错）。
交付成果
FPGA UDP/IP 协议解析 Verilog 模块、FIFO 跨时钟域代码、校验逻辑代码、仿真测试用例、配置帧解析模块、错误统计接口。

组员 5：FPGA 股票技术指标硬件并行运算开发
主要工作任务
1. 对照上位机 Python 源码，在 Verilog 中硬件复刻所有技术指标算法。
2. 实现硬件并行计算：MA5、MA20、MA60EMA 均线、RSI、MACD、布林带、ATR、成交量比率。
3. 完成浮点数转定点数量化设计，做截位、舍入、精度保留，保证 FPGA 运算结果与 Python 软件结果误差最小。
4. 采用流水线、并行架构设计，提升 FPGA 高速运算性能，满足高速交易要求。
5. 把运算完成的各项技术指标值输出给评分决策模块。
6. 配合组员 8 完成 Modelsim 仿真，逐波形比对 Python 理论值，修正算法逻辑。
7. 参与系统联调，修正指标运算偏差、数据时序匹配问题。
8. 【新增】 支持参数动态配置（MA 周期、RSI 周期可由上位机修改）。
9. 【新增】 定点运算加入舍入处理，降低系统误差。
交付成果
FPGA 全套技术指标运算 IP 核、定点数转换设计文档、算法对照说明、仿真波形文件、可配置参数版本指标模块。

组员 6：FPGA 评分算法、交易决策与 SFP 光口结果回传
主要工作任务
1. 硬件复刻 Python 中calculate_score百分制评分规则，用 Verilog 实现打分逻辑。
2. 按照均线趋势、RSI 区间、MACD 金叉死叉、成交量比例四个维度进行硬件累加打分。
3. 根据总分划分五档投资建议：强烈买入、建议买入、观望、建议卖出、强烈建议卖出。
4. 按照组员 2 约定的 UDP 通信协议，将评分、各项指标、交易信号封装成回传 UDP 数据包。
5. 开发 FPGA SFP 光口 UDP 发送模块，把结果数据包通过光口回传给上位机。
6. 完成模块独立逻辑调试、配合 Modelsim 仿真验证打分与封包逻辑。
7. 联调保证回传数据包格式正确、无错位、无丢失。
8. 【新增】 支持决策阈值可配置（上位机下发修改）。
9. 【新增】 回传包中加入状态字、错误计数、链路状态。
交付成果
FPGA 硬件评分与决策模块、UDP 结果封包发送模块、交易信号逻辑代码、仿真验证文件、可配置决策阈值版本、系统状态回传模块。
